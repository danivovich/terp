* Terp

   =terp= is a toy, lisp-like language.

   Currently implemented:
     + integers
     + basic arithmetic (=+=, =-=, =*=, =/=)
     + booleans (=#t=, =#f=)
     + =if then else= conditionals

** Usage
   #+BEGIN_SRC elixir
     iex> "(+ 5 3)"
     ...> |> Terp.eval()
     8

     iex> "(* 2 4 5)"
     ...> |> Terp.eval()
     40

     iex> "(* 2 4 (+ 4 1))"
     ...> |> Terp.eval()
     40

     iex> "(if #t (* 5 5) (+ 4 1))"
     ...> |> Terp.eval()
     25

     iex> "(if #f (* 5 5) (+ 4 1))"
     ...> |> Terp.eval()
     5
   #+END_SRC
